# CheckPoint 02 — Front-End Design

> Atividade avaliativa (CheckPoint 02) da disciplina de **Front-End Design** — 1º semestre de 2026. O objetivo é criar uma página web utilizando apenas HTML e CSS, com tema relacionado à Inteligência Artificial, aplicando estruturação semântica, layout com Flexbox e estilização com variáveis CSS.

---

## 📌 Descrição do projeto

A página foi desenvolvida com o tema **Claude (Anthropic)** e é composta por três seções principais + um rodapé:

- **Hero** — apresentação do Claude com título, parágrafo introdutório, imagem e dois botões de navegação.
- **Áreas de atuação** — quatro cards abordando a Anthropic, a IA Constitucional, os modelos disponíveis (Opus, Sonnet e Haiku) e os principais recursos da plataforma.
- **Sobre** — explicação mais detalhada das capacidades do Claude, com lista de características e imagem complementar.
- **Rodapé** — identificação dos integrantes do grupo com nome e RM.

---

## 🎨 Design

A identidade visual segue uma paleta minimalista de três cores:

| Variável      | Cor       | 
|---------------|-----------|
| `--laranja`   | `#DA7756` | 
| `--preto`     | `#1A1A1A` | 
| `--branco`    | `#F5F0EB` |

---

## 🔗 Navegação

A página conta com âncoras internas para navegação entre as seções:

- `#hero` → Seção inicial
- `#container` → Áreas de atuação
- `#sobre` → Sobre o Claude

---

## 🛠️ Tecnologias utilizadas

- **HTML5** — marcação semântica com seções bem definidas
- **CSS3** — variáveis CSS, Flexbox, unidades dinâmicas (`dvh`)
- **Google Fonts** — tipografias Cormorant (títulos) e Rubik (corpo)
